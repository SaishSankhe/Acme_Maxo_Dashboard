
# Xbiome coding challenge

A dashboard to visualise Acme and Maxo data.


## Tech Stack

**Client:** React

**Server:** Node, Express

  
## Run Locally

Clone the project

```bash
  git clone https://github.com/SaishSankhe/Xbiome_Coding_Challenge/
```

Go to the project directory

```bash
  cd Xbiome_Coding_Challenge
```

Install dependencies for server and client both

```bash
  cd client && npm install
```

```bash
  cd server && npm install
```

Start the server, in server as well as client

```bash
  cd server && npm start
```

```bash
  cd client && npm start
```
